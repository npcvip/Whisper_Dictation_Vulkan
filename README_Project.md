# 🎤 Whisper Dictation

**Whisper Dictation** — это локальное десктопное приложение для голосового ввода текста с использованием модели Whisper (OpenAI) и аппаратного ускорения через Vulkan.

### ✨ Возможности

* **Голосовой ввод в любое приложение** (Word, браузер, блокнот, IDE) по нажатию средней кнопки мыши.
* **Два режима работы**:

  * «Запись → анализ» — запись аудио, затем распознавание.
  * «Потоковый» — разбивка на чанки с постепенной вставкой текста.
* **Мгновенное распознавание**: модель Whisper загружается в память GPU через `whisper-server` (один раз при старте) — нет задержек на повторную загрузку.
* **Резервный режим** через `whisper-cli` (если сервер недоступен).
* **Гибкая настройка** частоты дискретизации (16 кГц / 48 кГц), усиления звука, горячих клавиш.
* **Управление через системный трей** (пауза, настройки, выход).
* **Поддержка любых Vulkan-совместимых GPU**: AMD, Intel (включая Arc), NVIDIA — благодаря сборке whisper.cpp с Vulkan.

### 🛠 Технологии

* Python (pyaudio, pyautogui, pynput, pystray, requests, tkinter)
* Whisper.cpp (whisper-cli / whisper-server)
* Модель: `ggml-large-v3-turbo-q8\_0.bin` (любая совместимая)
* Vulkan для аппаратного ускорения

### 🎯 Для кого

* Для быстрого ввода текста голосом без использования облачных сервисов.
* Для людей с ограниченной подвижностью рук.
* Для разработчиков, которые хотят интегрировать голосовой ввод в свои рабочие процессы.

### 🚀 Быстрый старт

Подробная инструкция по установке и настройке — в файле `README.md` (будет добавлен позже). Основные шаги:

1. Установить Python и зависимости (`pip install -r requirements.txt`).
2. Скачать whisper-cli и whisper-server с поддержкой Vulkan.
3. Поместить модель Whisper в папку `models/`.
4. Запустить `start\_whisper.bat`.
