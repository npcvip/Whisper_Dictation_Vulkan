# Whisper Dictation

**Локальный голосовой ввод текста с использованием Whisper и Vulkan.**

Whisper Dictation — десктопное приложение для Windows, которое распознаёт вашу речь и вставляет текст в любое приложение. Работает полностью офлайн, с аппаратным ускорением через Vulkan (GPU).

## Возможности

- **Полностью офлайн** — никаких облачных сервисов, всё работает локально
- **GPU-ускорение** — Vulkan для быстрой обработки на видеокарте
- **Два режима работы:**
  - **record_then_analyze** — запись → стоп → распознавание
  - **streaming** — распознавание в реальном времени по паузам речи
- **Два метода транскрипции:**
  - **whisper-cli** — локальный процесс, запускается для каждого файла
  - **whisper-server** — локальный HTTP-сервер, модель всегда в GPU, мгновенная генерация
- **VAD (Voice Activity Detection):**
  - **Внешний VAD (Ten VAD)** — определяет паузы в режиме стриминга
  - **Внутренний VAD (Silero)** — убирает тишину из аудио перед распознаванием
- **Горячие клавиши** — средняя кнопка мыши или кастомная комбинация
- **Иконка в трее** — всегда доступна, пауза, настройки, выход
- **Распознавание аудиофайлов** — WAV, MP3, FLAC, OGG, M4A, WMA и другие (автоматическая конвертация)
- **Шумоподавление при распознавании файлов** — high_pass (убрать гул) и low_pass (убрать шипение)

## Системные требования

- Windows 10/11 (64-bit)
- Python 3.10+
- GPU с поддержкой Vulkan (NVIDIA, AMD, Intel)
- Микрофон

## Быстрый старт

### Голосовой ввод

1. Запустите программу — иконка появится в трее
2. Нажмите среднюю кнопку мыши (или кастомную клавишу) — начало записи
3. Говорите
4. Отпустите кнопку — текст будет вставлен в активное приложение

### Распознавание аудиофайлов

1. Нажмите правую кнопку мыши на иконке в трее → "Выбрать аудиофайл"
2. Выберите файл (WAV, MP3, FLAC, OGG, M4A, WMA)
3. (Опционально) Включите шумоподавление:
   - ☐ Убрать гул (низкие частоты) — high_pass фильтр
   - ☐ Убрать шипение (высокие частоты) — low_pass фильтр
4. Нажмите "Распознать"
5. Нажмите "▶ Прослушать" для проверки качества шумоподавления

## Режимы работы

### Режим 1: record_then_analyze (Запись → Распознавание)

Классический режим: нажали — говорим — отпустили — текст вставлен. Подходит для коротких фраз и предложений.

### Режим 2: streaming (Стриминг)

Распознавание в реальном времени: программа слушает, определяет паузы речи и автоматически отправляет чанки на распознавание. Подходит для длинных текстов.

**Как работает стриминг:**
1. Весь аудио постоянно накапливается в буфер (речь + паузы)
2. Внешний VAD (Ten VAD) определяет паузы
3. Когда обнаружена пауза и накоплено достаточно аудио (≥ 3 сек) — чанк отправляется на распознавание
4. Отправка асинхронная — запись не останавливается
5. Внутренний VAD (Silero) убирает тишину из аудио перед распознаванием

## Настройки

### Движок

| Параметр | Описание |
|----------|----------|
| Устройство | GPU (быстро) / CPU (медленно) |
| Метод транскрипции | whisper-cli / whisper-server |
| Путь к модели | Файл модели Whisper (.bin) |
| Индекс GPU | 0 = первая GPU, 1 = вторая GPU |

### Звук

| Параметр | Диапазон | По умолчанию | Описание |
|----------|----------|-------------|----------|
| Режим работы | record_then_analyze, streaming | — | Режим записи |
| Частота дискретизации | 16000, 48000 | 16000 | Гц |
| Gain (усиление) | 1.0–10.0 | 2.0 | Усиление микрофона |
| Vad Threshold (внутренний) | 0.0–1.0 | 0.5 | Чувствительность Silero VAD |
| Vad Threshold (внешний) | 0.3–0.7 | 0.5 | Чувствительность Ten VAD |
| Min Silence Duration | 0.3–2.0 | 0.7 | Пауза для фиксации конца речи (сек) |
| Min Speech Duration | 0.1–1.0 | 0.25 | Мин. длительность речи (сек) |
| Min Chunk Duration | 0.5–5.0 | 3.0 | Мин. длительность чанка для отправки (сек) |

### Горячие клавиши

- **Средняя кнопка мыши** — по умолчанию (нажатие = старт, отпускание = стоп)
- **Кастомная комбинация** — Ctrl+Shift+A и другие

### Распознавание аудиофайлов

| Параметр | Диапазон | По умолчанию | Описание |
|----------|----------|-------------|----------|
| Убрать гул | 40–200 Гц | 80 Гц | High-pass фильтр (низкие частоты) |
| Убрать шипение | 4000–16000 Гц | 8000 Гц | Low-pass фильтр (высокие частоты) |
