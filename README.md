# 🎤 Whisper Dictation

**Локальный голосовой ввод текста с использованием Whisper и Vulkan.**

Whisper Dictation — десктопное приложение для Windows, которое распознаёт вашу речь и вставляет текст в любое приложение. Работает полностью офлайн, с аппаратным ускорением через Vulkan (GPU).

**Автор:** Anton Kuznetsov | **GitHub:** https://github.com/npcvip/Whisper_Dictation_Vulkan | **License:** MIT | **Created:** 2026

---

## ✨ Возможности

- **Полностью офлайн** — никаких облачных сервисов, всё работает локально
- **GPU-ускорение** — Vulkan для быстрой обработки на видеокарте (NVIDIA, AMD, Intel)
- **Два режима работы:**
  - **record_then_analyze** — запись → стоп → распознавание
  - **streaming** — распознавание в реальном времени по паузам речи
- **Два метода транскрипции:**
  - **whisper-cli** — локальный процесс, запускается для каждого файла
  - **whisper-server** — локальный HTTP-сервер, модель всегда в GPU, мгновенная генерация
- **VAD (Voice Activity Detection):**
  - **Внешний VAD (Ten VAD)** — определяет паузы в режиме стриминга
  - **Внутренний VAD (Silero)** — убирает тишину из аудио перед распознаванием
- **Выбор языка распознавания** — ru, en, auto и ещё ~100 языков
- **Горячие клавиши** — средняя кнопка мыши или кастомная комбинация
- **Иконка в трее** — всегда доступна, пауза, настройки, выход
- **Распознавание аудиофайлов** — WAV, MP3, FLAC, OGG, M4A, WMA (автоматическая конвертация)
- **Шумоподавление** — high_pass (убрать гул) и low_pass (убрать шипение)
- **Таймаут транскрипции** — защита от зависания (10 мин, можно отключить)
- **Защита от двойного запуска** — только одна копия программы

## 🖱️ Быстрый старт

### Голосовой ввод

1. Запустите программу — иконка появится в трее (синий круг = активна)
2. Нажмите **среднюю кнопку мыши** (или кастомную клавишу) — начало записи
3. Говорите
4. Отпустите кнопку — текст будет вставлен в активное приложение

### Распознавание аудиофайлов

1. Правый клик на иконке в трее → **"Выбрать аудиофайл"**
2. Выберите файл (WAV, MP3, FLAC, OGG, M4A, WMA)
3. (Опционально) Включите шумоподавление:
   - ☐ Убрать гул (низкие частоты) — high_pass фильтр
   - ☐ Убрать шипение (высокие частоты) — low_pass фильтр
4. Нажмите **"Распознать"**
5. Нажмите **"▶ Прослушать"** для проверки качества
6. Скопируйте результат кнопкой **"Скопировать всё"**

## 💡 Советы по выбору модели

По умолчанию программа поставляется с моделью **ggml-large-v3-turbo** (Q5 или Q8 квантизация, зависит от релиза).

- **Q5** — быстрее, меньше памяти. Рекомендуется для встроенных GPU (Intel UHD/Iris)
- **Q8** — выше качество, больше памяти. Рекомендуется для дискретных GPU (NVIDIA, AMD, Intel Arc)

**Как подобрать оптимальные параметры:**
1. Откройте окно распознавания аудиофайла (правый клик → "Выбрать аудиофайл")
2. Выберите любой аудиофайл
3. Нажмите "Распознать" — после распознавания отобразится **время обработки**
4. Попробуйте разные модели и методы (whisper-cli / whisper-server) — выберите оптимальный вариант по скорости и качеству

**Скачать модели:** https://huggingface.co/ggerganov/whisper.cpp/tree/main

## 🗣️ Режимы работы

### Режим 1: record_then_analyze (Запись → Распознавание)

Классический режим: нажали — говорим — отпустили — текст вставлен. Подходит для коротких фраз и предложений.

**Как работает:**
1. Нажатие горячей клавиши → запись аудио
2. Отпускание → остановка записи
3. Транскрипция через whisper-cli или whisper-server
4. Вставка текста через буфер обмена (Ctrl+V)

### Режим 2: streaming (Стриминг)

Распознавание в реальном времени: программа слушает, определяет паузы речи и автоматически отправляет чанки на распознавание. Подходит для длинных текстов.

**Как работает стриминг:**
1. Весь аудио постоянно накапливается в буфер (речь + паузы)
2. **Внешний VAD (Ten VAD)** определяет паузы
3. Когда обнаружена пауза и накоплено достаточно аудио (≥ 3 сек по умолчанию) — чанк отправляется на распознавание
4. Отправка **асинхронная** — запись не останавливается
5. **Внутренний VAD (Silero)** убирает тишину из аудио перед распознаванием
6. Принудительная отправка каждые 10 секунд (защита от зависания)
7. Отпускание клавиши → остаток буфера → транскрипция

## 🎙️ VAD — два уровня

### Внутренний VAD (Silero, whisper-cli/server)

- **Зачем:** убирает тишину из аудио ПЕРЕД распознаванием, чтобы Whisper не тратил время на пустые участки
- **Как работает:** флаги `--vad --vad-model` передаются в whisper-cli и whisper-server
- **Настройка:** Vad Threshold (0.0–1.0) — чувствительность
  - Меньше (0.3) = чувствительнее, больше аудио отправляется
  - Больше (0.8) = строже, только уверенную речь
- **По умолчанию:** включён, threshold = 0.5

### Внешний VAD (Ten VAD, стриминг)

- **Зачем:** определяет паузы в режиме стриминга, чтобы разбивать поток речи на чанки
- **Как работает:** анализирует аудио в реальном времени, определяет начало/конец речи
- **Настройки:**
  - **Vad Threshold** (0.3–0.7) — чувствительность к речи
  - **Min Silence Duration** (0.3–2.0 сек) — пауза для фиксации конца речи
  - **Min Speech Duration** (0.1–1.0 сек) — минимальная длительность речи для учёта
  - **Min Chunk Duration** (0.5–5.0 сек) — минимальная длительность чанка для отправки
- **По умолчанию:** включён в режиме стриминга

Whisper поддерживает ~100 языков. **Совет:** если говорите по-русски (иногда вставляя английские слова) — используйте `ru`, это даст максимальное качество (+5-15% точности). Если аудио содержит много английского — используйте `auto`. |
| **Индекс GPU** | `0` = первая GPU, `1` = вторая GPU. Определяется автоматически (приоритет: дискретная GPU) |

| **Метод транскрипции** | `whisper-cli` — прямой вызов для каждого файла; `whisper-server` — локальный HTTP-сервер, модель всегда в GPU, мгновенная генерация |

## 🎧 Распознавание аудиофайлов

Откройте окно: правый клик на иконке в трее → **"Выбрать аудиофайл"**.

### Возможности

- **Поддержка форматов:** WAV, MP3, FLAC, OGG, M4A, WMA (автоматическая конвертация в WAV 16kHz mono)
- **Шумоподавление:**
  - **Убрать гул (низкие частоты)** — high_pass фильтр, 40–200 Гц, по умолчанию **100 Гц**
  - **Убрать шипение (высокие частоты)** — low_pass фильтр, 4000–16000 Гц, по умолчанию **11000 Гц**
- **Прослушивание** — кнопка **"▶ Прослушать"** открывает обработанный файл в плеере Windows
- **Таймаут транскрипции** — защита от зависания (10 минут). Кнопка **"Таймаут: ВКЛ/ВЫКЛ"** в правом нижнем углу:
  - **ВКЛ** — если whisper-cli зависнет, транскрипция прервётся через 10 минут
  - **ВЫКЛ** — без ограничения (для очень длинных файлов)
- **Время обработки** — отображается после текста (например, "Время обработки: 12 сек")

### Совет

Используйте окно распознавания аудиофайла для тестирования разных моделей и настроек — время обработки поможет подобрать оптимальный вариант.

## 🔄 Автоматический перезапуск сервера

При использовании метода **whisper-server** программа автоматически перезапускает сервер при изменении:

- **GPU device index** — смена видеокарты
- **Модели** — смена файла модели
- **Устройства** — переключение GPU ↔ CPU
